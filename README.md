Dependency and Installation Issues
-------------------------------------
In order to run the project successfully the following dependencies are needed
```
OpenNI.net.dll
OpenNI64.dll
XnVNITE.net.dll
Microsoft.Kinect.dll (Kinect SDK V2 dll)
```
